Unassign Documents

 Top  Previous  Next

If you no longer need to have a document assigned to you, select the document and select the Unassign command. This cancels the assignment and the Owner field is cleared.

 

When you unassign a document which is awaiting review by you, the document's status changes to Review. When you unassign a document which is awaiting review by someone else, the document's status changes to Hold. If you unassign a document in a library, its status changes to IN.

 

Any user can unassign a document that was assigned to them. The document can also be unassigned by the last owner (the person who did the assigning.) Only users with workflow manager library access rights or administrators can unassign documents that were assigned to other users.
