Column Sets

 Top  Previous  Next

Column sets allow the user to use, save and edit a particular set of columns in Adept's windows. See the results area topic for details.

 

Note: Currently, Column Sets can only be created within the Adept Desktop Client application. To have a Column Set available to you when using other Adept Client applications, the COlumn Set must first be created in the Adept Desktop Client application by yourself or by another user who shares the Column Set with you.
