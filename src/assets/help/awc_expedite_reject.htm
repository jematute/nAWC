Expedite Reject

 Top  Previous  Next

When the workflow step has the option enabled to "Allow Expedite Reject at this step by Workflow Manager,"and if you have at least Workflow Manager rights for the selected document, you can issue the Expedite Reject command. This command allows you to move the document any number of steps backward in the workflow including rejecting it to the first step.

 

Multiple documents

You can select multiple documents and Expedite Reject them to a step in a workflow ONLY if all the selected documents are in the same workflow and the same library.  If they are not, the action will fail.

 

Expediting can only occur to steps outside the range of steps that ALL the selected documents are currently on.  For example, if 3 documents from the same workflow are selected and they are on steps 3, 5, and 7 respectively, they can only be expedite rejected to steps less than 3.
