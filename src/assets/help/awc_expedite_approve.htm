Expedite Approve

 Top  Previous  Next

When the workflow step has the option enabled to "Allow Expedite Approve at this step by Workflow Manager" and if you have at least Workflow Manager rights for the selected document, then you can issue the Expedite Approve command. This command allows you to move the document any number of steps forward in the workflow including approving it at the last step and returning it to the library.

 

The dialog box may offer an option to create a version. This option is only applicable to certain libraries and only when expediting to Final Approval.

 

Multiple documents

You can select multiple documents and Expedite Approve them to a step in a workflow ONLY if all the selected documents are in the same workflow and the same library.  If they are not, then your step options will be limited to the <Final Approve> step as that is the only workflow step that is common to all selected documents.

 

Note: Expediting can only occur to steps outside the range of steps that ALL the selected documents are currently on.  For example, if 3 documents from the same workflow are selected and they are on steps 3, 5, and 7 respectively, they can only be Expedite Approved to steps greater than 7.
