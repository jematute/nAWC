Microsoft Word: Workstation PlugIn Options

Set the Workstation PlugIn options for your Word integration by selecting the PlugIn Workstation command on the Tools tab of the ribbon, then select Word from the pull down list.

 

Use Watcher

The "watcher" program should only be turned on if requested to do so by Synergis technical support staff. The Synergis Watcher is a debug program used to help identify issues related to the proper integration between Adept and an application. Be sure to turn off the "watcher" program when done, as it will affect performance if left running.
