PlugIn for PDF Extraction

 Top  Previous  Next

When you load and enable this PlugIn, Adept will be able to extract Form Text Fields, Properties and Thumbnails from PDF documents. Select the Configure PlugIns command to open the Administration dialog.  The Administration dialog allows control of what is extracted under Options.

 

When checking the option to extract thumbnails, a dialog will be presented where you define the quality of the thumbnail. This setting impacts the size of the image stored in the database and the time needed to perform extraction. Setting this above 50% may cause excessive database size resulting in longer database backups and longer extraction time. It is recommended to set this to 40% or less.

 

The bottom portion of the dialog allows you to map the properties in the external documents with the extraction fields in Adept.

 

As with other PlugIns, you will need to set up Extraction Fields and map them to the properties. See your Adept administrator for configuration details.

 

Note: Only PDF's that are not secured can be extracted. Secure PDF's prevent the PlugIn from extracting any information successfully.
