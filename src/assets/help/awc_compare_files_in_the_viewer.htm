Compare Documents in the Viewer

 Top  Previous  Next

For some types of documents, such as AutoCAD dwg, you can use the viewer's compare feature to highlight the differences between two documents. To compare two drawings, load them into the viewer by selecting the documents and select the View Document command on the Home or View tab. Then from the viewer's Analysis menu, select compare.

 

Note: See the topic View Document: Library or Work In Progress to learn how to load multiple versions of the same document.

 

When you select Compare from the Analysis menu, if you have two drawings loaded into the viewer, the two images are shown side by side and the differences are highlighted in a third window. If you only have one drawing loaded, or more than two, then the viewer will use the drawing currently loaded in the window and will ask you to select the other one from Adept's interface, such as from the Search Results or FileGuide.

 

Note: If the Compare Documents dialog does not appear in the foreground, you may need to alt-tab to bring it forward.
