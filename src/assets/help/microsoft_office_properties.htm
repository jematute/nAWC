PlugIn for Microsoft Word

 Top  Previous  Next

Use the PlugIn for Microsoft Word to read or write properties, Form Fields, and Content Controls from Word 2003 or above documents. To use this PlugIn you must also have Microsoft Word installed on your workstation.

 

Note: The PlugIn for Microsoft Word cannot Extract properties from MS Word versions previous to 2003.

 

Note: If you are using Microsoft Office 2010 or higher, you may need to unblock older versions of Microsoft Word. To check or change File Blocking settings in Microsoft Office, click the File Menu, then click Options. On the left side, click Trust Center, and in that dialog click File Block Settings. Unblock the versions of Word that you need to synchronize.

 

Feature description

Word documents contain some built in properties, such as Author and Subject. You can also create your own Custom Properties such as Client or Project. Word properties can be linked to form fields or Content Control fields inside Word.

When you Synchronize or Check In Microsoft Office documents, 2003 or greater, Adept checks to see if any properties match Extraction fields in your database. If there is a match between your Extraction fields and the Properties, Form Fields, and Content Control fields in the document, Adept copies the value out of the document into the database.

 

An effective way to use these properties is for your administrator to create template documents that contain predefined custom properties. When you create a new document with the Document New command, Adept creates a copy of the template. You can then add values to the properties in the document. When you check in the document, Adept will then automatically copy the values into the database.

 

Set Up

To set up the PlugIn, open the Administrator PlugIn dialog by selecting the Administer PlugIns command. Select Adept PlugIn for Word from the list of PlugIns then click Edit to open the PlugIn Editor. Check Load on Startup then click OK to close the dialog and enable the PlugIn. Click OK to close the Administer PlugIns dialog and install the PlugIn.

 

Select the Configure PlugIns command to open the Administration dialog for this PlugIn. Check the Extensions and Options as appropriate. When you have this PlugIn installed, you can set up your Extraction fields in Adept to link to Properties, Form Fields, and Content Controls in Word. Then, as with other Extraction fields, you can write (update) values from Adept to the Word document, or read values from the Word document to Adept.

 

Note: It is recommended that you map to the form field or the Content Control field instead of the custom property. Mapping to the custom property and performing an Update Document will break any link that the custom property might have to a form field or Content Control field.

 

As with other PlugIns, you will need to set up Extraction Fields and map them to the properties. See your Adept administrator for configuration details.
