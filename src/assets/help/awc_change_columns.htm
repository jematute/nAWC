Add Columns, Change Columns

 Top  Previous  Next

The data associated with each document (linked or unlinked) stored in Adept is displayed in columns in the results area. To specify which columns are displayed in these windows,select the three line icon (hamburger) on the title bar at the far right side. You can drag and drop columns from one side to the other, adding or removing them from the columns displayed in the results area.

 

Note: The number of columns directly impacts performance when refreshing the window.

 

Note: Columns can be dragged and dropped within the column header to reorder the columns as displayed.
