Document Status: New

 Top  Previous  Next

A New status indicates a document that has never been checked in to a Library. New documents only appear in your Work Area. Adept creates a temporary copy of the Data Card. The information from this Data Card is copied into the database when you check in the new document.

 

New never appears as a status entry in a Library.

 

If you rename a New document to have a name that appears in the database, the status is changed to Duplicate.
