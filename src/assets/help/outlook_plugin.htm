Overview

The Outlook PlugIn can be used in conjunction with the “Outlook to Adept” Add-in. Emails and their attachments can be saved from Outlook to Adept. When the email is saved as a record in Adept, properties from the emails such as To, From, and Subject can be automatically inserted into Adept extraction fields using the Outlook PlugIn.

 

What Properties can be extracted from an Outlook Email

Here is a list of all the Properties available in an Outlook email.

 

•Billing Info

•CC

•Category

•Companies

•Creation Time (Date and Time)

•Creation Date (Date only portion of above) – To map to Adept Date type field

•Deferred Delivery Time (Date and Time)

•Deferred Delivery Date (Date only portion of above) – To map to Adept Date type field

•Flag Due By (Date and Time)

•Flag Due By Date (Date only portion of above) – To map to Adept Date type field

•Flag Request

•Last Modified Time (Date and Time)

•Last Modified Date (Date only portion of above) – To map to Adept Date type field

•Received By

•Received on Behalf of Name

•Received Time (Date and Time)

•Received Date (Date only portion of above) – To map to Adept Date type field

•Reply Recipient Names

•Sender email address

•Sender Name

•Sent On (Date and Time)

•Sent On Date (Date only portion of above) – To map to Adept Date type field

•Sent on Behalf of

•Subject

•To

•*Task Completed (Date and Time)

•*Task Completed Date (Date only portion of above) – To map to Adept Date type field

•*Task Start (Date and Time)

•*Task Start Date (Date only portion of above) – To map to Adept Date type field

•*Task Subject

 

Note: Microsoft Office 2007 and greater provides a set of properties for “Task” related emails, marked in the above list by an *.

 

Note: In the CC and To fields only the displayed name is found and not necessarily the expanded email address.

 

Note: This PlugIn cannot extract BCC.

 

Note: This PlugIn cannot use the Update Document feature to push values from Adept back to the Outlook Email.

 

Note: All of the Date/Time type of properties offer two different ways to access them, one with just the date and the other with the date and time. The “date only” properties will only be allowed to map into an Adept Date Field.

 

Note: When a field is mapped to an Adept field the string will get truncated if the Adept field is not large enough to capture the entire value.

 

Configure

To configure this PlugIn, you will need to map Outlook Email Properties to Adept Extraction Fields. First select the Configure PlugIns command…

 

The resulting dialog displays a list of all the Adept Extraction Fields in the left column, and a list of all the Microsoft Outlook Email Property Fields in the right column.

 

To map an Extraction field to an Outlook property, select the Extraction field from the column on the left. The name of the Extraction field will be displayed above the middle column. Now review the Email Properties in the column on the right. Drag one or more properties from this list and drop it on to the middle column.

 

When you are finished mapping properties, click OK.
